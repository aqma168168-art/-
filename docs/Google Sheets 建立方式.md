# Google Sheets 建立方式

執行 `initSheets()` 會自動建立以下所有工作表。本文件說明各工作表的欄位用途。

---

## 系統設定

| 設定項目 | 設定值 | 說明 |
|----------|--------|------|
| 每桶供貨價 | 1275 | 中廚供貨給攤位的每桶價格（元） |
| 大碗售價 | 80 | 攤位大碗對外售價（元） |
| 小碗售價 | 60 | 攤位小碗對外售價（元） |
| 老闆PIN | （請自行設定） | 老闆登入 PIN，建議 6 位數字 |
| 系統名稱 | 中央廚房管理系統 | |
| 時區 | Asia/Taipei | |

> 修改供貨價或售價後，損益計算自動使用新值，不需改程式。

---

## 攤位設定

| 欄位 | 說明 | 範例 |
|------|------|------|
| 攤位編號 | 唯一識別碼，不可重複 | S001 |
| 攤位名稱 | 攤位名稱 | 信義攤位 |
| 啟用 | 是否啟用（TRUE / FALSE） | TRUE |
| 月租金 | 月租金（元） | 8000 |
| 月固定成本 | 每月固定支出，例如水電、人事 | 2000 |
| 備註 | 備註 | |

**新增攤位：** 在此工作表末尾新增一列，攤位編號格式建議 S + 3 位數字（如 S004）。

## 庫存基準

設定各品項的初始庫存與安全庫存。

| 欄位 | 說明 |
|------|------|
| 品項編號 | 對應配料設定的配料編號 |
| 品項名稱 | 品項名稱 |
| 單位 | 計量單位 |
| 初始庫存 | 系統啟用時的起始庫存數量 |
| 安全庫存 | 低於此值時系統發出警示 |
| 備註 | 備註 |

> 目前庫存 = 初始庫存 + 所有入庫 − 所有出庫（從庫存異動表計算）

---

## 重要注意事項

1. **工作表名稱（分頁標籤）請勿修改**，程式會以此查找工作表。
2. **第一列（標題列）請勿修改**，欄位順序與程式對應。
3. **攤位編號、配料編號一旦使用就不可更改**，歷史資料用這些 ID 串接。
4. `啟用` 欄位值為 Google Sheets 布林值 `TRUE` / `FALSE`，不是文字。
